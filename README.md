# Assignment Pinball 

## Roll The Ball (R.T.B)
It is a pinball game done for the first assignment of the physics subject. 

## Installation
Decompress the .zip file in any directory and execute the executable inside the decompressed folder.

#### Controls:
**A** - Activate right flipper

**D** - Activate left flipper

**SPACE** - Activate pusher (launch ball).

**TAB** - Restart the game on the game over screen.

**F1** - Debug Mode.
